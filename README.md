# ✈️📊 Airline Operations & Performance Dashboard 📊✈️

Excited to present my interactive **Airline Operations & Performance Analysis Dashboard**, built using real-world aviation traffic data to uncover operational trends, passenger movement, and cargo performance insights! 🚀

---

## 📌 Project Overview

This project analyzes airline passenger and cargo operations to understand:

* Traffic trends over time
* Airline performance comparison
* Passenger and cargo distribution
* Regional and terminal activity patterns

The dashboard transforms raw aviation datasets into actionable business insights using **Excel data preparation** and **Tableau visualization**.

---

## 🧹 Data Preparation (Excel)

Before visualization, extensive data cleaning and transformation were performed:

* Added and formatted readable **date columns**
* Converted data types for accurate analysis
* Replaced blank values with meaningful categories (e.g., *Unknown*)
* Checked and validated missing values
* Created multiple analytical **Pivot Tables** for insight discovery
* Applied proper number formatting and sorting for clarity

### Key Analytical Pivot Studies

⭐ **Top Airlines Analysis**
Identified airlines carrying the highest passenger and cargo volume.

⭐ **Domestic vs International Distribution**
Compared traffic share across geographic categories.

⭐ **Trend Over Time Analysis**
Grouped data by Months and Years to analyze long-term traffic growth.

⭐ **Aircraft Type Analysis**
Examined cargo contribution from Passenger, Freighter, and Combi aircraft.

⭐ **Airport Flow Analysis**
Studied activity type distribution across airport operations.
